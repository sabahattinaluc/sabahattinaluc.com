## Overview

A refined, editorial-style personal portfolio for a designer-founder. The aesthetic evokes the calm confidence of a high-end design studio — warm cream backgrounds, gold accents, elegant serif display type paired with clean sans-serif body text. Nothing screams; everything communicates.

## Colors

- **Primary (#2D2D2D):** Deep charcoal — headlines, nav, primary text
- **Secondary (#7A7570):** Warm taupe — supporting text, metadata
- **Accent (#D4A853):** Antique gold — CTAs, highlights, decorative elements
- **Neutral (#F5F0E8):** Warm cream — page background
- **Surface (#FAF8F5):** Off-white — card/section backgrounds
- **Text Inverse (#F0EBE3):** Cream — text on dark backgrounds

## Typography

- **Display/Hero:** Cormorant Garamond 300 — elegant, editorial serif
- **Body/UI:** DM Sans 400/500 — clean, modern sans-serif
- **Hierarchy:** 3rem hero → 2.5rem h1 → 1.5rem h2 → 1rem body

## Layout

Single-page scrolling layout with four main sections: hero, about, selected work, and contact. Generous whitespace, max-width 960px centered, mobile-first responsive.

## Do's and Don'ts

- Do: Use generous whitespace to let content breathe
- Do: Keep CTAs minimal — gold accent, restrained use
- Don't: Clutter with too many elements per section
- Don't: Use more than two typefaces
